## 📑기능 목록
+ **기능 1. 구입 금액 입력받기**
  - [예외 처리] 1000원 단위 아닐 시 예외 처리

<br> 

+ **기능 2. 로또 발행**
    - 제공된 랜덤 API 사용해서 구현
    - 오름차순 정렬
    - 각 자리의 수는 1부터 45 사이의 수
    - 중복 불가

<br>

+ **기능 3. 당첨 번호 및 보너스 번호 입력받기**
    - 없는 쉼표로 구분
    - 각 자리의 수는 1부터 45 사이의 수
    - 중복 불가
    - [예외 처리] 형식에 맞지 않을 시 예외 처리

<br>

+ **기능 4. 당첨 통계 계산**
    - 당첨 내역 계산
    - 총 수익률 계산(소수점 둘째 자리에서 반올림)

<br>

+ **기능 5. 당첨 통계 출력**
    - 당첨 내역 출력
    - 총 수익률 출력
